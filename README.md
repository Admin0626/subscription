# 简订Pro - 订阅管理小程序

一个简洁好用的订阅管理微信小程序，帮助你追踪和管理每月的订阅支出。

## 功能特性

- **订阅管理**：添加、编辑、删除订阅服务
- **支出统计**：按月/按年查看订阅费用统计
- **即将过期提醒**：智能提醒即将到期的订阅
- **数据导出**：支持导出订阅账单
- **图标识别**：主流订阅服务自动匹配图标

## 页面说明

### 首页
- 显示月度/年度支出总览
- 即将过期订阅提醒
- 所有订阅列表

### 添加页
- 快速选择主流订阅服务（Netflix、Spotify、YouTube 等）
- 自定义添加订阅

### 统计页
- 本月/本年支出对比
- 分类占比饼图
- 订阅排行

### 设置页
- 数据导出
- 清空数据

## 技术栈

- 微信小程序原生开发
- WXML + WXSS + JavaScript

## 使用说明

1. 微信开发者工具导入项目
2. 选择 `subscription` 文件夹作为项目目录
3. 使用测试号或正式号体验

## 项目结构

```
subscription/
├── app.js          # 应用入口
├── app.json       # 应用配置
├── app.wxss      # 全局样式
├── theme.wxss     # 主题变量
├── assets/icons/   # 服务图标
├── pages/
│   ├── index/    # 首页
│   ├── add/     # 添加订阅
│   ├── stats/    # 统计页
│   ├── detail/   # 订阅详情
│   ├── edit/     # 编辑订阅
│   └── settings/ # 设置页
└── utils/        # 工具函数
```
