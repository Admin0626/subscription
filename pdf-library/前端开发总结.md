# PDF 图书分享平台 - 前端开发总结

## 项目概述

基于 Vue 3 + Vite + Pinia 开发的 PDF 图书分享平台（线上图书馆）前端应用。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | 3.x | 渐进式前端框架 |
| Vite | - | 构建工具 |
| Pinia | - | 状态管理 |
| Vue Router | - | 路由管理 |
| Axios | - | HTTP 请求 |

## 设计系统

### 色彩方案（暖色调大地色系）

| 颜色 | 值 | 用途 |
|------|------|------|
| 主色 | #78716C | 按钮、链接、强调（Stone） |
| 主色深 | #57534E | hover状态 |
| 主色浅 | #A8A29E | 背景、选中状态 |
| 辅助色 | #92400E | CTA按钮（Amber） |
| 背景 | #FAFAF9 | 页面背景（Warm） |
| 表面 | #FFFFFF | 卡片背景 |
| 文字 | #1C1917 | 主要文字 |
| 文字次 | #78716C | 次要文字 |
| 边框 | #E7E5E4 | 分割线、边框 |

### 字体

- 主字体：**Inter**
- 备选：-apple-system, BlinkMacSystemFont, Segoe UI, Roboto

### 设计特点

- 暖色调大地色系现代设计风格
- 8px 间距系统
- 圆角：6px / 8px / 10px / 14px / 16px
- 微阴影效果
- 平滑过渡动画

## API 对接

已在 `src/api/index.js` 中封装完整的 API 调用：

- `authApi` - 认证相关
- `userApi` - 用户资料
- `bookApi` - 图书操作
- `categoryApi` - 分类
- `tagApi` - 标签
- `favoriteApi` - 收藏
- `homeApi` - 首页数据
- `notificationApi` - 通知
- `adminApi` - 管理后台
